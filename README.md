# Ruby 2.3, 2.4, 2.5 or 2.6 with Node.js 11 LTS Dockerfile

A minimal Dockerfile based on Ruby (2.3, 2.4, 2.5 or 2.6) Dockerfile (regular, slim or alpine) with Node.js 11 LTS installed.

## What's included

* Ruby 2.3, 2.4, 2.5 or 2.6
* Node.js 11
* npm
* yarn
